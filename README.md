# Baller University Website

## 📌 Project Overview

This project is a fully responsive multi-page website for a fictional sports training academy called **Baller University**. The website promotes athlete development programs, events, and opportunities to get involved.

---

## 🌐 Pages Included

* Home
* Events
* Get Involved
* About
* Contact

---

## 🛠️ Techs Used

* HTML5
* CSS3
* Bootstrap 5.3
* Bootstrap Icons
* AOS (Animate On Scroll Library)
* JavaScript

---

## 🎨 Features

* Fully responsive design
* Animated sections using AOS
* Interactive forms (Get Involved & Contact)
* Embedded Google Map
* Consistent navigation and footer across all pages
* Custom CSS styling for branding

---

## 📱 Responsiveness

The website is fully responsive and works on:

* Mobile devices
* Tablets
* Desktop screens
